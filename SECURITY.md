\# Security Policy



\## Supported Versions



| Version | Supported |

| ------- | --------- |

| 1.0.0   | Yes       |



\## Responsible Use



WifiNetworkScanner is intended for local diagnostics on Wi-Fi networks owned by you or networks where you have explicit authorization to perform diagnostics.



Do not use this app to scan networks without permission.



\## Data Handling



WifiNetworkScanner does not send scan data to external servers in version 1.0.0.



The app stores scan history locally on the device and only shares TXT/CSV reports when the user explicitly chooses to export or share them.



\## Reporting a Vulnerability



If you find a security issue, please open a GitHub issue with:



\- App version.

\- Android version.

\- Device model.

\- Steps to reproduce.

\- Expected behavior.

\- Actual behavior.



Avoid posting sensitive network data, private IP lists, screenshots with personal information, passwords, tokens, or router credentials.



\## Scope



Security reports may include:



\- Unsafe file sharing.

\- Incorrect permission behavior.

\- Unexpected data exposure.

\- Crashes caused by malformed local network data.

\- Problems related to exported TXT/CSV files.



Out of scope:



\- Scanning networks without authorization.

\- Requests to bypass Android security restrictions.

\- Requests to hide scanning activity.
